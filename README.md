# TCP-Chat-App



## usage :
### first you should make them executable
`chmod +x server.py`
`chmod +x client.py`

### then you can run them
##### on server machine

`./server.py`

##### on client machine

`./ client.py`


### to send file

`send -f filename`

**you can run any number of client as the server is multithreaded**
